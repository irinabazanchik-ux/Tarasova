
# Book Tracker

**Автор:** Тарасова Дарья Андреевна

## 📋 Описание программы
**Book Tracker** — это десктопное приложение с графическим интерфейсом (GUI), предназначенное для ведения персонального списка прочитанной литературы. Программа помогает систематизировать библиотеку, отслеживать объем прочитанного и быстро находить нужные книги по критериям.

### Ключевые возможности:
 * **Управление списком:** добавление книг с указанием названия, автора, жанра и количества страниц.
 * **Динамическая фильтрация:** поиск по жанру или фильтрация книг по объему (минимальное количество страниц).
 * **Надежное хранение:** автоматическое сохранение данных в формат JSON — ваш список не пропадет после закрытия программы.
 * **Валидация данных:** встроенная проверка на пустые поля и корректность ввода числовых значений (страниц).
   
## 🛠 Технологический стек
 * **Язык программирования:** Python 3.10+
 * **Библиотека GUI:** Tkinter (стандартная библиотека Python)
 * **Формат данных:** JSON
 * **Система контроля версий:** Git
## 🚀 Инструкция по запуску
### 1. Предварительные требования
Убедитесь, что у вас установлен Python версии 3.7 или выше. Проверить версию можно командой:
```bash
python --version

```
### 2. Установка
Склонируйте репозиторий или скачайте архив с файлами:
```bash
git clone <ссылка_на_ваш_репозиторий>
cd book_tracker

```
### 3. Запуск приложения
Запустите основной файл скрипта:
```bash
python main.py

```
## 📖 Примеры использования

### Добавление книги
 1. Заполните поле **Название** (напр., *"451 градус по Фаренгейту"*).
 2. Укажите **Автора** (*Рэй Брэдбери*).
 3. Укажите **Жанр** (*Антиутопия*).
 4. Введите **Количество страниц** (*256*).
 5. Нажмите кнопку **"Добавить"**. Книга мгновенно появится в таблице снизу.

### Фильтрация списка
 * **По жанру:** Введите в блоке фильтрации слово *"Фантастика"*. Нажмите **"Применить"**. В списке останутся только книги этого жанра.
 * **По страницам:** Введите в поле "Мин. страниц" число *300*. Программа скроет все небольшие рассказы и покажет только объемные произведения.
 * **Сброс:** Для возврата к полному списку нажмите кнопку **"Сброс"**.

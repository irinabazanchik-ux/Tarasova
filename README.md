# Book Tracker

**Автор:** Иванов Иван

## Описание программы
Приложение для учета прочитанных книг. Позволяет сохранять названия, авторов, жанры и объем книг. Все данные автоматически сохраняются в файл JSON.

## Инструкция по запуску
1. Установите Python 3.x.
2. Скачайте файл `main.py`.
3. Запустите скрипт командой: `python main.py`.

## Примеры использования
* **Добавление:** Введите "Дюна", "Фрэнк Герберт", "Фантастика", "600" и нажмите "Добавить".
* **Фильтрация:** Введите в фильтре жанр "Фантастика" и нажмите "Применить".
* **Фильтр страниц:** Введите "200" в поле страниц, чтобы увидеть только объемные книги.
